PHP 安装 Xdebug 扩展来调试程序

发表于 2018-01-30 更新于 2021-07-01 分类于 php

概述

  • Xdebug Website
  • Xdebug GitHub

Xdebug 是 PHP 的一个扩展，以协助调试和开发。它包含一个 single step debugger 与 IDEs 一起使用；它升级了 PHP 的 var_dump() 函数；它为通知、警告、错误和异常添加了堆栈跟踪；它具有记录每个函数调用和变量赋值到磁盘的功能；它包含一个分析器；并且提供了与 PHPUnit 一起使用时的代码覆盖功能。

调试窗口说明：

  • Debugger：调试器，比如：跳过，单步调试等操作
  • Frames：运行的帧，当前项目调试时载入的关联程序
  • Variables：调试中的所有变量
  • Watches：监视器，自定义变量监视
